    • Apa Itu Pengolahan Data Perikanan dan Kelautan?

    Mata kuliah ini bertujuan untuk mengenalkan mahasiswa pada teknik-teknik pengolahan data perikanan dan kelautan menggunakan alat seperti CTD dan software-software yang berhubungan dengan kelautan.

    Software yang Digunakan

    Microsoft Excel

    SeaFET Data Processing

    Ocean Data View

    Notepad++

    Software yang digunakan meliputi SBE Data Processing untuk menganalisis data CTD, Ocean Data View (ODV) untuk visualisasi data, serta Microsoft Excel untuk pengolahan dasar. Setiap software ini memiliki peran unik dalam menghasilkan informasi yang bermanfaat.
